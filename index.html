📂 Как использовать в Obsidian
  1. Скачайте файлы (или ZIP, или один файл)
  2. Скопируйте их в папку вашего Vault (например Vault/Ledger/)
  3. В Obsidian установите плагин Dataview (Settings → Community Plugins)
  4. Создайте сводную заметку и вставьте DQL-запрос — пример будет в файле _README.md

Для работы нужен Client ID из Google Cloud Console. Один раз настраивается, потом работает автоматически. Как получить?

🗄 Резервные копии — автоматически, последние 14 дней
Раз в день перед первым изменением программа сохраняет копию всех данных в этом браузере. Отдельные копии создаются перед синхронизацией, импортом и восстановлением.
—

📘 Как получить Client ID

1. Открой Google Cloud Console в своём Google-аккаунте.

2. Создай новый проект — слева сверху выпадающий список → «Новый проект» → имя «Расчётная программа».

3. Меню ☰ → API и сервисы → Библиотека → найди Google Drive API → нажми «Включить».

4. Меню ☰ → API и сервисы → Экран согласия OAuth:
• Тип — External
• Имя — любое
• Email — свой
• Сохраняй до конца
• Вверху — «Опубликовать приложение»

5. Меню ☰ → API и сервисы → Учётные данные → «+ Создать» → «Идентификатор клиента OAuth»:
• Тип приложения — Веб-приложение
• Разрешённые источники JavaScript — добавь URL, где открываешь файл:
http://localhost
https://localhost
А если хостишь на GitHub Pages / своём сайте — добавь и этот URL.

6. После создания скопируй Client ID (заканчивается на .apps.googleusercontent.com) и вставь в программу.

Важно: Google OAuth не работает при открытии файла двойным кликом (file://). Нужен http://localhost или https-хостинг.
